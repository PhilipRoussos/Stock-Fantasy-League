Stock Fantasy League is a comprehensive real-time multiplayer application where users create private leagues, invite friends via QR codes or Bluetooth, chat with competitors, and manage portfolios based on live stock market data.

For detailed information and installation instructions, please refer to the **`README.pdf`** located in the `README` folder.

## API Keys Setup

Before running the application, you need to configure your own API keys. 
Please open the **`lib/config.dart`** file and insert your active keys for:
* **Finnhub API**
* **Google Gemini AI**
